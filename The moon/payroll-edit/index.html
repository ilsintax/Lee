1. 엑셀 데이터 붙여넣기

엑셀에서 변환할 전체 범위(제목행 포함)를 복사(Ctrl+C)하여 아래에 붙여넣기(Ctrl+V) 하세요.

2. 제목행 선택 및 직렬화

데이터의 제목행(Header)을 클릭하세요. 선택한 제목과 바로 윗줄의 구분(제목)을 기준으로 데이터를 변환합니다.

3. 데이터 정렬 확인 및 필터링 기준 선택

직렬화가 완료되었습니다! 금액(지급페이 등)에 해당하는 열의 제목을 클릭하세요. 해당 열에 데이터가 없는(빈칸인) 행은 자동으로 제외됩니다.

4. 1차 분류 (다중 일괄 배분)

항목을 체크박스로 선택하고 사업장으로 한 번에 이동시키세요. (번호 윗줄의 내용으로 자동 그룹화 되었습니다)

미분류 데이터 (0)
분류할 사업장

5. 2차 분류 (자동 매칭 및 수정)

1차 분류에서 매칭된 이름과 주민등록번호를 바탕으로 남은 데이터가 자동 분류되었습니다. 내역을 확인하고 필요시 수정하세요.

최종 분류 완료

모든 데이터가 성공적으로 분류되었습니다.
